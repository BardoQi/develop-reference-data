5.25. Patch-2.5.4

Patch 软件包包含一个根据补丁文件来修改原文件的程序。补丁文件通常是用 diff 程序创建的。

预计编译时间： 少于 0.1 SBU
所需磁盘空间： 1.6 MB

5.25.1. 安装 Patch

为编译 Patch 做准备：

./configure --prefix=/tools

编译软件包：

make

这个软件包没有附带测试程序。

安装软件包：

make install

关于这个软件包的详细资料位于节 6.48.2, Patch 的内容
