5.23. M4-1.4.4

M4 包含一个宏处理器。

预计编译时间： 少于 0.1 SBU
所需磁盘空间： 3 MB

5.23.1. 安装 M4

为编译 M4 做准备：

./configure --prefix=/tools

编译软件包：

make

要测试结果，请运行：make check 。

安装软件包：

make install

关于这个软件包的详细资料位于节 6.16.2, M4 的内容
