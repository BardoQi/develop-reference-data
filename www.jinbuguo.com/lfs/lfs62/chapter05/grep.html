5.21. Grep-2.5.1a

Grep 可以按指定的匹配模式搜索文件中的内容。

预计编译时间： 0.1 SBU
所需磁盘空间： 4.8 MB

5.21.1. 安装 Grep

为编译 Grep 做准备：

./configure --prefix=/tools \
    --disable-perl-regexp

配置选项的含义：

--disable-perl-regexp

这个选项确保 grep 程序不连接可能在宿主系统上存在的 PCRE(Perl 兼容正则表达式)库，因为进入 chroot 环境后，它就不能使用了。

编译软件包：

make

要测试结果，请运行：make check 。

安装软件包：

make install

关于这个软件包的详细资料位于节 6.37.2, Grep 的内容
