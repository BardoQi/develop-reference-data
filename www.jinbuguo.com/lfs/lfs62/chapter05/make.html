5.24. Make-3.80

Make 软件包包含一个编译软件包的程序。

预计编译时间： 0.1 SBU
所需磁盘空间： 7.8 MB

5.24.1. 安装 Make

为编译 Make 做准备：

./configure --prefix=/tools

编译软件包：

make

要测试结果，请运行：make check 。

安装软件包：

make install

关于这个软件包的详细资料位于节 6.44.2, Make 的内容
