6.44. Make-3.80

Make 自动地确定一个大型程序的哪些片段需要重新编译，并且发出命令去重新编译它们。

预计编译时间： 0.1 SBU
所需磁盘空间： 7.8 MB

6.44.1. 安装 Make

为编译 Make 做准备：

./configure --prefix=/usr

编译软件包：

make

要测试结果，请运行：make check 。

安装软件包：

make install

6.44.2. Make 的内容

安装的程序： make

简要描述
