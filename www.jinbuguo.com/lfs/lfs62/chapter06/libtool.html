6.21. Libtool-1.5.22

GNU libtool 是一个通用库支持脚本，将使用动态库的复杂性隐藏在统一的、可移植的接口中。

预计编译时间： 0.1 SBU
所需磁盘空间： 16.6 MB

6.21.1. 安装 Libtool

为编译 Libtool 做准备：

./configure --prefix=/usr

编译软件包：

make

要测试结果，请运行：make check 。

安装软件包：

make install

6.21.2. Libtool 的内容

安装的程序： libtool, libtoolize
安装的库： libltdl.{a,so}

简要描述

libtool

提供通用的库编译支持。
libtoolize

提供了一种标准方式来将 libtool 支持加入到一个软件包中
libltdl

隐藏 dlopening 库的复杂细节
