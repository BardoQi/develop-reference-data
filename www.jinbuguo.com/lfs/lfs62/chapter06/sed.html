6.20. Sed-4.1.5

Sed 是一个流编辑器。

预计编译时间： 0.1 SBU
所需磁盘空间： 6.4 MB

6.20.1. 安装 Sed

为编译 Sed 做准备：

./configure --prefix=/usr --bindir=/bin --enable-html

新配置选项的含义：

--enable-html

This builds the HTML documentation.

编译软件包：

make

要测试结果，请运行：make check 。

安装软件包：

make install
