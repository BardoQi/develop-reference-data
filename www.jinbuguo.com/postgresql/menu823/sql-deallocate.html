DEALLOCATE

名称

DEALLOCATE -- 删除一个预备语句

语法

DEALLOCATE [ PREPARE ] name

描述

DEALLOCATE 用于删除前面编写的预备语句。如果你没有明确删除一个预备语句，那么它将在会话结束的时候被删除。

有关预备语句的更多信息。参阅 PREPARE

参数

PREPARE

这个关键字总被忽略

name

将要删除的预备语句

兼容性

SQL 包括一个 DEALLOCATE 语句，但它只是用于嵌入式 SQL 。
