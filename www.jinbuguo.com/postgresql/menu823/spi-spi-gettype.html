SPI_gettype

函数名

SPI_gettype -- 返回声明字段的数据类型名

原型

char * SPI_gettype(TupleDesc rowdesc, int colnumber)

描述

SPI_gettype 返回声明字段的数据类型名字的一份拷贝。如果你不再需要它了，你可以使用 pfree 释放名字的拷贝。

参数

TupleDesc rowdesc

输入行描述

int colnumber

字段编号(从 1 开始记)

返回值

声明的字段的数据类型名，如果错误，则为 NULL 。错误时，SPI_result 设置为 SPI_ERROR_NOATTRIBUTE
