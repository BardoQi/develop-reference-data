SPI_cursor_move

函数名

SPI_cursor_move -- 移动一个游标

原型

void SPI_cursor_move(Portal portal, bool forward, long count)

描述

SPI_cursor_move 忽略游标中的一些航。这个函数等效于 SQL 命令 MOVE

参数

Portal portal

包含游标的入口

bool forward

向前移动时为真，向后移动时为假

long count

移动的最大行数
