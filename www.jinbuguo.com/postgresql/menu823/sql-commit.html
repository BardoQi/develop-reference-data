COMMIT

名称

COMMIT -- 提交当前事务

语法

COMMIT [ WORK | TRANSACTION ]

描述

COMMIT 提交当前事务。所有事务的更改都将为其它事务可见，而且保证当崩溃发生时的可持续性。

参数

WORK
TRANSACTION

可选关键字，没有任何作用。

注意

使用 ROLLBACK 语句退出事务。

在一个事务外部发出 COMMIT 不会有问题，但是将产生一个警告信息。

例子

提交当前事务以让所有变更永久化：

COMMIT;

兼容性

SQL 标准只声明了 COMMIT 和 COMMIT WORK 两种形式。否则完全兼容。
