43.37. pg_group

pg_group 的存在是为了向下兼容：它模拟一个存在于 PostgreSQL 版本 8.1 之前的系统表。它显示所有标记为不是 rolcanlogin 的角色的名字和成员，这就是近似于用做组的那些角色的集合了。

表43-37. pg_group 字段

grosysidoidpg_authid.oid组的 ID
grolistoid[]pg_authid.oid一个数组，包含这个组里面所有角色的 ID
