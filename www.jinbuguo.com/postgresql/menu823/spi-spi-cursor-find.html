SPI_cursor_find

函数名

SPI_cursor_find -- 用名字寻找并执行一个现存的游标

原型

Portal SPI_cursor_find(const char * name)

描述

SPI_cursor_find 通过名字寻找一个现存的入口。主要用于解析一个其它函数返回的文本的游标名字。

参数

const char * name

入口的名字

返回值

指向指定名称的入口的指针，如果没有找到就是 NULL
