SPI_exec

函数名

SPI_exec -- 执行一个读/写命令

原型

int SPI_exec(const char * command, long count)

描述

SPI_exec 和 SPI_execute 一样，只是相当于后者的 read_only 参数总是 false

参数

const char * command

包含需要执行的命令的字符串

long count

处理或者返回的最大行数

返回值

参见 SPI_execute
