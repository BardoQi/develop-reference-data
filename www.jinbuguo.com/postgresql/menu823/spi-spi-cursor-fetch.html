SPI_cursor_fetch

函数名

SPI_cursor_fetch -- 从一个游标里抓取一些行

原型

void SPI_cursor_fetch(Portal portal, bool forward, long count)

描述

SPI_cursor_fetch 从一个游标里抓取一些行。这个函数等效于 SQL 命令 FETCH

参数

Portal portal

包含游标的入口

bool forward

向前抓取时为真，向后抓取时为假

long count

要抓取的最大行数

返回值

成功时，像 SPI_execute 那样设置 SPI_processed 和 SPI_tuptable
