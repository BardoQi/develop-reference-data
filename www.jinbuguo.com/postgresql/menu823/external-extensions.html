H.3. 扩展

PostgreSQL 从开始就设计成可以扩展的。因此，加载到数据库的扩展可以像它们被打包在数据库里一样。和源代码一起发布的 contrib/ 目录里面有大量这种扩展。该目录下的 README 文件包含一个概述。它们包含转换工具、全文索引、XML工具、额外的数据类型和索引方法等等。其它扩展是独立开发的，比如 PostGIS 。甚至 PostgreSQL 的复制方案也是在外部开发的。比如 Slony-I 是一个流行的主/从复制方案，它就是独立在核心项目之外开发的。

PostgreSQL 还有好几种管理工具可以用。最流行的是 pgAdmin III ，以及还有几种商业版本。
