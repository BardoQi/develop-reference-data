ALTER LANGUAGE

名称

ALTER LANGUAGE -- 修改一个过程语言的定义

语法

ALTER LANGUAGE name RENAME TO newname

描述

ALTER LANGUAGE 修改一个语言的定义。目前唯一的功能就是重命名语言。只有超级用户可以重命名一个语言。

参数

name

语言的名字

newname

语言的新名字

兼容性

SQL 标准里没有 ALTER LANGUAGE 语句。
