本章描述 SQL 的语法。这些内容是理解随后各章的基础，那些章里面将详细介绍 SQL 命令如何用于定义和修改数据。

我们也建议那些已经很熟悉 SQL 的用户仔细阅读本章，因为有一些规则和概念在 SQL 数据库之间并不一致，或者是有些东西是 PostgreSQL 特有的。
