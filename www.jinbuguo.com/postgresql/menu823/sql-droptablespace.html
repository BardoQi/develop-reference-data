DROP TABLESPACE

名称

DROP TABLESPACE -- 删除一个表空间

语法

DROP TABLESPACE [ IF EXISTS ] tablespacename

描述

DROP TABLESPACE 从系统里删除一个表空间。

一个表空间只能由其所有者或者超级用户删除。在删除一个表空间之前，表空间里面不能有任何数据库对象。即使当前数据库里面已经没有任何对象在使用这个表空间了，也有可能有其它的数据库对象存留在这个表空间里。

参数

IF EXISTS

如果指定的表空间不存在，那么发出一个 notice 而不是抛出一个错误。

tablespacename

表空间的名字

注意

DROP TABLESPACE 不能出现在事务块内部。

例子

从系统里删除表空间 mystuff ：

DROP TABLESPACE mystuff;

兼容性

DROP TABLESPACE 是 PostgreSQL 扩展
