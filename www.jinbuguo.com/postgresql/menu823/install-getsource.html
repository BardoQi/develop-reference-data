14.3. 获取源码

PostgreSQL 8.2.3源代码可以从匿名FTP站点 ftp://ftp.postgresql.org/pub/source/v8.2.3/postgresql-8.2.3.tar.gz 获得。其它下载选项可以在我们的站点 http://www.postgresql.org/download/ 找到。在你获取文件之后，解压缩：

gunzip postgresql-8.2.3.tar.gz
tar xf postgresql-8.2.3.tar

这样将在当前目录创建一个目录 postgresql-8.2.3 ，里面是 PostgreSQL 源代码。进入这个目录完成安装过程的其它步骤。
